Fonction DBSCAN calculant les centres d'interets.

L'algorithme DBSCAN est un algorithme de partitionement basé sur la densité des clusters. Il prend deux parametres espilon, la distance minimale entre les point d'un cluster et min_point le nombre de point minimum dans un cluster.
